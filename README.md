# Baptiste Williams - Digital Resume

A static website hosted on **AWS S3** and masked with a custom domain from **No-IP**.  
This project showcases my resume, biography, portfolio, and includes extra credit features like video integration.

---

## Features
- **Biography**: Background and academic journey
- **Resume**: Education, experience, and skills
- **Portfolio**: GitHub projects + embedded YouTube video
- **Contact Form**: Redirects to a thank-you page
- **Bootstrap Framework**: Responsive, modern design
- **Custom Domain Masking**: S3 endpoint masked with No-IP

---

##  Live Demo
- **No-IP Domain**: http://baptiste-resume.hopto.org/  
- **S3 Endpoint**: http://baptiste-digital-resume-2.s3-website-us-east-1.amazonaws.com
